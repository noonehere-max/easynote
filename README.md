# Easy Note

Una aplicación web de notas simple, bonita y funcional. Guarda tus notas directamente en el navegador usando `localStorage`.

## Características

- Crear, editar y eliminar notas
- Guardado automático en el navegador
- Búsqueda de notas en tiempo real
- Tema claro y oscuro
- Copiar contenido al portapapeles
- Diseño responsive y moderno
- Atajos de teclado:
  - `Ctrl + N`: Nueva nota
  - `Ctrl + D`: Eliminar nota activa

## Cómo usar

1. Abre el archivo `index.html` en tu navegador favorito.
2. Haz clic en **Nueva** para crear una nota.
3. Escribe el título y el contenido.
4. ¡Listo! Tus notas se guardan automáticamente.

## Tecnologías

- HTML5
- CSS3
- JavaScript vanilla
- Font Awesome (iconos)
- Google Fonts (Inter)

## Almacenamiento

Las notas se guardan en el `localStorage` de tu navegador. No se envían a ningún servidor.
